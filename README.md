# Prompt Injection Lab

## Overview

This repository demonstrates common prompt injection attacks against Large Language Models (LLMs) and explores basic defense strategies.

The goal is to understand how malicious prompts can manipulate AI systems and how safer prompting and validation techniques can reduce risks.

---

## Topics Covered

- Prompt Injection
- Jailbreak Prompts
- System Prompt Protection
- Input Validation
- Safe Prompt Design
